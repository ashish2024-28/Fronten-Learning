Three ways to add CSS to HTML

  1. Inline Css <- rare use , means line may.
    2. style="color: yellowgreen; background-color: blueviolet;"
  2. Internal Css <- less use , means use in Html's head.
    3. in head
    <style> #Internal{ color: rgb(10, 233, 114); background-color: blueviolet; } </style>
  3. External Css <- More use , means use in css file
    4. #External{ color: red; background-color: yellow; }
